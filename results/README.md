# Results

`sample_run/` contains a tiny synthetic output-format example. It is not a benchmark result.

`b0-openai-gpt-5-6-luna-smoke/` and `b1-openai-gpt-5-6-luna-smoke/` are curated 20-task hosted pipeline checks. `b0-vs-b1-openai-gpt-5-6-luna-smoke/` contains their reproducible paired comparison. Each directory labels its scope and limitations explicitly.

Full local traces belong under `results/local/` or `results/runs/` and are ignored. Publish a result set only after checking dataset redistribution terms, removing credentials or private provider metadata, and linking the exact configuration and software revision.
